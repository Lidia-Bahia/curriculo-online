# 📄 Currículo Online

Este repositório contém o código-fonte do meu currículo online, desenvolvido como uma página web para apresentar minha formação acadêmica, habilidades e projetos.

O site foi construído utilizando tecnologias fundamentais de desenvolvimento web e está hospedado através do **GitHub Pages**.

[**Acesse o currículo aqui**](https://lidia-bahia.github.io/curriculo-online/)

## ⚙️ Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript  

## 💻 Estrutura do projeto

- [`index.html`](https://github.com/Lidia-Bahia/curriculo-online/blob/main/index.html) — estrutura principal da página  
- [`formacao.html`](https://github.com/Lidia-Bahia/curriculo-online/blob/main/formacao.html) — informações sobre formação acadêmica  
- [`portfolio.html`](https://github.com/Lidia-Bahia/curriculo-online/blob/main/portfolio.html) — projetos e trabalhos desenvolvidos  
- [`contato.html`](https://github.com/Lidia-Bahia/curriculo-online/blob/main/contato.html) — página de contato  
- [`contato.js`](https://github.com/Lidia-Bahia/curriculo-online/blob/main/contato.js) — interatividade da página de contato  
- [`style1.css`](https://github.com/Lidia-Bahia/curriculo-online/blob/main/style1.css) — estilos e layout do site  

## 📌 Objetivo

Disponibilizar uma versão online do meu currículo e reunir, de forma acessível, informações sobre minha formação, habilidades e projetos.
